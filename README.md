
## Sneakers API

- https://sneakers-api-cmkf.onrender.com

#### get all sneakers

```bash
  GET /
```

#### get sneaker with ID

```bash
  GET /get-sneaker/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `number` | **Required**. id of sneaker to fetch |
